## RealEstateCRM Backend

Backend API built with **NestJS** and **TypeScript**.

### Scripts

- **Development**: `npm run start:dev`
- **Build**: `npm run build`
- **Production start**: `npm run start` (after build)

### Running locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the dev server:
   ```bash
   npm run start:dev
   ```
3. Open `http://localhost:3000/health` to check that the API is running.

This is a minimal baseline. You can now add modules for authentication, properties, contacts, and deals as separate NestJS modules.
